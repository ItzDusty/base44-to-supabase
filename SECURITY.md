# Security Policy

## Supported versions

This project is currently in early development. Only the latest version is supported.

## Reporting a vulnerability

Please open a GitHub issue with the **"security"** label.

If your report includes sensitive details, you can instead open an issue with minimal
information and ask for a private channel to share reproduction steps.

## Scope

- This repository (CLI, codemods, adapters)
- Generated migration output is user-specific and not covered by this policy
